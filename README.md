# Playwright E2E Framework

![Playwright Tests](https://github.com/josemendezqa/playwright-sauce-e2e/actions/workflows/playwright.yml/badge.svg)

End-to-end test automation framework built with Playwright and TypeScript.

## 🚀 Features

- Page Object Model (POM)
- Custom fixtures
- Test steps for improved reporting
- HTML reports with Playwright
- GitHub Actions CI integration

## 📁 Project Structure

fixtures/
pages/
tests/
.github/workflows/

## ▶️ Run testsv

npx playwright test


👀 Run in headed mode

npx playwright test --headed

📊 View report

npx playwright show-report

⚙️ CI

Tests run automatically on every push using GitHub Actions.
